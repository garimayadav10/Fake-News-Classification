# Fake News Classification using Machine Learning

This project classifies news articles as Fake or Real using NLP and Machine Learning techniques.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- TF-IDF Vectorizer

## How to Run
1. Install requirements
   pip install -r requirements.txt

2. Train model
   python train_model.py

3. Predict news
   python app.py

## Output
- ❌ Fake News
- ✅ Real News
